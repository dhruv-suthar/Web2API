## Quick Start

```bash
npm install
npm run dev
```

## Configuration

Create `.env.local` file:

```bash
NEXT_PUBLIC_API_URL=http://localhost:3001/api
```

For production:

```bash
NEXT_PUBLIC_API_URL=https://your-backend.up.railway.app/api
```

## Tech Stack

- **Framework**: Next.js 16 + React 19
- **Styling**: Tailwind CSS 4
- **State**: React Query
- **Icons**: Lucide
